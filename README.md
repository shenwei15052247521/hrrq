# 青州市民卡-食堂端系统说明

## 使用技术&框架

- SpringBoot2.0.4
- Maven3项目构建
- Redis缓存服务器
- Nginx静态资源代理
- MySQL5.7数据库
- RabbitMQ消息队列
- Vsftp文件服务器

## 项目结构说明

- qzsmk-canteen(**_pom_**)
    - auth(**_jar_**)
    - common(**_jar_**)
    - core(**_jar_**)
    - parent(**_pom_**)
